Installation / Usage
--------------------
Install Xi-CAM:

Follow the guide [here](https://xi-cam2.readthedocs.io/en/latest/install.html)

Install lbl_ir:

    $ git clone https://lchen2ir@bitbucket.org/berkeleylab/lbl_ir.git
    $ cd lbl_ir
    $ pip install -e .

Clone the repo:

    $ git clone https://github.com/lchen23/Xi-cam.BSISB.git
    $ cd Xi-CAM.BSISB
    $ pip install -e .
    $ cd .. 
    $ xicam
